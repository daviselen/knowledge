# Knowledge

The canonical knowledge base for our agency.

This repository contains the documentation, standards, processes, and institutional knowledge that help our team work effectively. It is designed for both human readers and AI systems that assist with research, writing, planning, and decision-making.

## Principles

- This repository is the source of truth whenever practical.
- Keep information close to where it is maintained.
- Prefer many focused documents over a few large ones.
- Link related documents instead of duplicating information.
- When in doubt, document it.

## Repository Structure

- `/about` — Who we are and how we're organized
- `/clients` — Client information
- `/services` — What we offer
- `/processes` — How we work
- `/playbooks` — Best practices
- `/standards` — Writing, design, development, etc.
- `/templates` — Reusable documents
- `/glossary` — Agency terminology

## Contributing

If you discover something that should be documented, please submit a pull request.

Prefer:
- One topic per file
- Clear headings
- Short paragraphs
- Links to related documents
